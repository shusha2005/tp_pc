В рамках тестирования были проверены основные функции backend-части веб-приложения PC Club Booking.

1. Авторизация

* Проверена регистрация пользователя
* Проверен вход в систему с корректными данными
* Проверена обработка ошибки при неверных данных

2. Работа с клубами

* Получение списка клубов через API
* Проверка корректного ответа сервера (status 200)
* Проверка наличия данных в ответе

3. Работа с ПК

* Получение списка ПК
* Фильтрация ПК по `club_id`
* Проверка корректности ответа и наличия данных
